# Resumer Challenge

Another approach of the cloud resumer challenge.

## Tech Stack

|Name|Description|
|----|-----------|
|FlexBox model|Define the CSS design and layout|
|LocalStorage|Store the count of visit in the browser|
|Unicorn|Run the web server|
|FastAPI|Provide the api endpoint|


## Roadmap
### Step 1 : Frontend development
- HTML : build the resumer
- CSS  : design the layout
- JS   : capture the visit and display the number of visit
